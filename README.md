# 🃏 算24点游戏 - 可爱卡通版

一个支持1-6名玩家的线下算24点网页游戏，具有可爱卡通风格UI和完整的积分排行系统。

## 🎮 游戏特色

- **多人游戏**：支持1-6名玩家同时参与
- **可爱卡通UI**：精美的卡通风格界面设计
- **智能积分**：正确答案基础分 + 时间奖励积分
- **实时排行**：游戏过程中实时显示排行榜
- **完整流程**：玩家加入 → 游戏轮次 → 最终排行榜

## 🚀 如何开始

### 方法一：使用Python启动本地服务器
```bash
cd demo
python -m http.server 8000
```
然后在浏览器中访问：`http://localhost:8000/24-points-game.html`

### 方法二：直接在浏览器中打开
直接用浏览器打开 `demo/24-points-game.html` 文件即可开始游戏。

## 🎯 游戏规则

1. **玩家加入**：输入姓名并选择可爱的头像（最多6人）
2. **游戏开始**：系统随机发放4张牌（A=1, J=11, Q=12, K=13）
3. **限时答题**：30秒内输入算式，使用 +, -, ×, ÷ 运算符
4. **答案验证**：系统自动验证答案是否等于24
5. **积分奖励**：
   - 正确答案：100分基础分 + 时间奖励（剩余秒数×2）
   - 错误答案：0分
6. **多轮游戏**：共10轮，结束后显示最终排行榜

## 🎨 界面特色

- **渐变背景**：梦幻紫色渐变背景
- **卡片动画**：发牌时的翻转动画效果
- **粒子特效**：背景浮动粒子装饰
- **状态指示**：不同颜色显示玩家答题状态
- **响应式设计**：适配手机和电脑端

## 🏆 积分系统

- **基础分数**：正确答案获得100分
- **时间奖励**：剩余时间×2分（越快答对分数越高）
- **累计积分**：多轮游戏累计总分
- **正确次数**：统计每位玩家答对题目数

## ⌨️ 操作说明

- **添加玩家**：输入姓名 → 选择头像 → 点击"添加玩家"
- **开始游戏**：至少1名玩家即可开始
- **输入答案**：在对应输入框中输入算式（如：`6*4`、`8+8+8`等）
- **下一轮**：每轮结束后点击"下一轮"继续
- **结束游戏**：10轮后或随时可结束查看最终排行榜

## 🔧 自定义配置

可在代码中修改以下参数：
- `totalRounds`: 游戏总轮数（默认10轮）
- `timeLeft`: 每轮答题时间（默认30秒）
- `baseScore`: 基础分数（默认100分）
- `timeBonus`: 时间奖励倍数（默认×2）
